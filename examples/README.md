# Smart BLE - 示例代码

## 目录说明

本目录包含教学用的示例代码，帮助开发者理解 BLE 开发的各个方面。

## 示例列表

### basic-scan/
基础扫描示例 - 最简单的 BLE 扫描实现。

**学习目标**：
- 如何初始化蓝牙适配器
- 如何开始和停止扫描
- 如何处理扫描结果
- 如何正确释放资源

**支持平台**：uni-app, Flutter, Android, iOS

---

### connect-and-communicate/
连接和通信示例 - 完整的设备连接、读写通信流程。

**学习目标**：
- 如何连接 BLE 设备
- 如何发现服务和特征值
- 如何读取特征值
- 如何写入特征值
- 如何启用通知
- 如何处理连接断开

**支持平台**：uni-app, Flutter, Android, iOS

---

### custom-protocol/
自定义协议示例 - 实现 ESP32 灯控协议的完整示例。

**学习目标**：
- 如何设计 BLE 协议
- 如何编码/解码数据
- 如何实现可靠的数据传输
- 如何处理协议错误

**支持平台**：uni-app, Flutter, Android, iOS

---

## 使用方法

每个示例目录都包含：
- `README.md` - 详细说明文档
- `src/` - 示例源代码
- `docs/` - 相关教程链接
