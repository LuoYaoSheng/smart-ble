## 写作规范

### 统一格式

- **标题**：主标题 + 副标题
- **导语**：引用或问题引入
- **正文**：分章节，使用 emoji 图标
- **代码**：使用代码块，标注语言
- **截图**：配图说明，增强可读性
- **结尾**：行动号召（Star、Fork、分享）

### 统一 Footer

```
---

*文末扫码关注公众号，回复「BLE」获取项目链接和技术交流群*

*本文为原创技术文章，转载请注明出处*
```

```

002-开发路线图.md
003-Bug修复记录-v1.0.1.md
004-新功能-OTA升级.md
...
```
