# 🌟 [EP01] 入门局：下载指南与神级深色模式

> 这是《Smart BLE 全栈实战指南》的第一集。本集绝不碰一行代码，只教你如何白嫖。

当您受够了各大应用商店里充斥着十年未更新、且界面极其简陋的“串口小助手”时，您一定会爱上这款作品：**Smart BLE Toolkit+**。

## 一、一键下载防坑指北

由于这款软件是我们刚刚开源编造的大型跨端项目，尚未花大价钱去买微软或者苹果的企业签名，因此在首次运行时可能会受到系统的层层阻挠。

### 1.1 下载途径
不用去找网盘，您直接跳转到本项目的 [🚀 GitHub Release 页](#)。里面罗列了当周由云端集群自动打包好的最热乎的文件。挑选您对应的系统平台：
- **手机扫码最方便**：提供 Android (APK) 以及可内测的 iOS (TestFlight)。
- **桌面端**：对于 Windows 选择 `.msi` 结尾即可。对于 MacOS 则是 `.dmg`。

### 1.2 小白防杀毒拦截必学操作
1. **Windows**：双击安装后，屏幕变蓝，中心弹框？别怕，点击 `更多信息 (More Info)`，再点击 `仍要运行 (Run Anyway)`，瞬间释放蓝牙猛兽。
2. **Mac**：打开显示“已损坏”？不要移入废纸篓！直接找到图标，用键盘按死 `Control` 键不松开，然后鼠标点击它选“打开”，就可以越权使用了。
3. **Android**：如果打开时它向您索取**「位置权限」**，这是正当防卫要求。因为无论是 iOS 还是 Android，“扫描周围低功耗蓝牙 (BLE)”必须伴随 GPS 定位权限。如果拒绝，就会产生千古未解之锁：“为什么我的 APP 始终搜不到蓝牙”。

## 二、从浅入深的雷达图扫描

进入软件后，第一眼震撼您的必须是它的 UI 界面：没有乱糟糟的发报框，只有清晰的科技动效。

1. **发起搜索**：点击屏幕正中心的巨大雷达按钮。列表会像瀑布一样刷出周围几十台开启了蓝牙的设备。
2. **连接**：相中了任何一个 MAC 地址或设备名？轻轻点击，软件会自动发起协议握手进入到真正的详情操控面板。不用担心连接慢卡死，我们有强大的多线程并发机制！

## 三、高颜值才是第一生产力：暗黑模式

这是本集我们最想传达的卖点。在任意一个有设置按钮的地方，点击**“暗黑模式”**。

在极其统一的代码拉通力量下，您的 iPhone / 安卓机 以及 您的 Windows 宽屏电脑上的每一个卡片、文字甚至发着蓝光的电弧 SVG 图像，将会完美对齐变换。带着这套暗黑引擎去给老板演示，您的身价必定暴涨。
