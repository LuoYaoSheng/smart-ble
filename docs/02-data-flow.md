```

       │
       ▼
┌──────────────────┐
│  设备缓冲区       │  累积所有发现的设备
│  deviceBuffer    │
└────────┬─────────┘
         │

┌──────────────────┐

         │
         ▼

         │
         ▼

```

### 4.2 数据转换

```javascript
// ArrayBuffer 转 Hex
function ab2hex(buffer) {
    const hexArr = Array.from(new Uint8Array(buffer))
        .map(bit => ('00' + bit.toString(16)).slice(-2))
    return hexArr.join('')
}

// ArrayBuffer 转 UTF-8
function ab2utf8(buffer) {
    const decoder = new TextDecoder('utf-8')
    return decoder.decode(buffer)
}
```

### 5.2 数据编码

```javascript
// UTF-8 字符串转 ArrayBuffer
function stringToArrayBuffer(str) {
    const buffer = new ArrayBuffer(str.length)
    const dataView = new DataView(buffer)
    for (let i = 0; i < str.length; i++) {
        dataView.setUint8(i, str.charCodeAt(i))
    }
    return buffer
}

// HEX 字符串转 ArrayBuffer
function hexToArrayBuffer(hex) {
    const buffer = new ArrayBuffer(hex.length / 2)
    const dataView = new DataView(buffer)
    for (let i = 0; i < hex.length; i += 2) {
        dataView.setUint8(i / 2, parseInt(hex.substr(i, 2), 16))
    }
    return buffer
}
```

## 12. 错误处理流程

### 12.1 统一错误处理

```
┌─────────────────────────────────┐
│  API 调用                        │
└──────┬──────────────────────────┘
       │
       ▼
┌─────────────────────────────────┐
│  检查返回结果                    │
│  res.errCode / err.errCode       │
└──────┬──────────────────────────┘
       │
       ├── 成功 ──> 继续正常流程
       │
       ▼ 失败
┌─────────────────────────────────┐
│  记录错误日志                    │
│  addLog('error', errorMessage)  │
└──────┬──────────────────────────┘
       │
       ▼
┌─────────────────────────────────┐
│  显示用户提示                    │
│  - Toast (一般错误)             │
│  - Modal (需用户操作)           │
└─────────────────────────────────┘
```

### 12.2 常见错误处理

| 错误场景 | 日志类型 | 用户提示 |
|----------|----------|----------|
| 蓝牙未开启 | error | 请先开启系统蓝牙 |
| 权限被拒绝 | error | 需要蓝牙/定位权限 |
| 设备连接超时 | error | 连接超时，请重试 |
| 写入失败 | error | 数据写入失败 |
| 服务不存在 | error | 设备服务异常 |
