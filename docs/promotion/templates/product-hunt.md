### Product Description (介绍)

```
Smart BLE is a professional Bluetooth Low Energy debugging tool with 8+ platform implementations.

Key Features:
- 8+ platforms: uni-app, Flutter, Electron, Tauri, macOS, Avalonia, and more
- Fully open-source: frontend, desktop, and ESP32 firmware
- Complete ESP32 firmware included
- MIT license - free for commercial use

Perfect for:
- Bluetooth device developers
- Cross-platform app developers
- Embedded engineers
- BLE learners
```

### Launch Tips (发布建议)

1. **发布时间**：美国时间 12:01 AM PT（太平洋时间凌晨）
2. **发布前准备**：
   - 准备好猎人账号
   - 关注 Product Hunt 上相关产品的发布者
   - 提前几天在社交媒体预热

3. **发布当天**：
   - 早上 8-10 点 PT 是黄金时间
   - 及时回复评论
   - 在社交媒体分享链接

4. **第一句话很重要**：
   - 简洁有力，说明解决的问题
   - 示例："Finally, an open-source BLE tool that actually works across all platforms!"

5. **讨论区**：
   - 准备一些常见问题的回答
   - 引导用户去 GitHub Star
   - 收集反馈意见
