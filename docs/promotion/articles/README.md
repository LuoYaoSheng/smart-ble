# Smart BLE 文章索引

这些文章不只是“推广内容”，也是 `smart-ble` 作为多实现、强教学项目的重要知识资产。

建议把这组文章理解成 3 条并行内容线：

- **产品线**：帮助新用户理解 Smart BLE 是什么、为什么值得看
- **教学线**：帮助开发者按主题学习 BLE、平台差异和硬件实践
- **演进线**：记录项目为什么这样设计、为什么保留多实现和历史分支
