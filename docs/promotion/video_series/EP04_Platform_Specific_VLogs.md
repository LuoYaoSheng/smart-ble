# [实况分镜脚本] 独研网课专区：五大平台实装 VLog 系列

在播完了面向大众用户的三大系列剧场后。我们需要针对垂直细分流量（也就是这五大开发社区的小圈子），做专门的圈层传播。这就需要录制所谓的**第一视角网课 Vlog**：即把你的电脑屏幕原封不动给这群人看。
这套扩展包剧本不需要什么酷炫剪辑，主打“真实有效，看完就能在电脑上跑起来”，建议分别截取上传到特定的论坛或技术专栏！

---

## 📹 [VLog 01] Flutter 专场：你所不知道的蓝牙动态授权天坑
* **主旨对象**：Flutter 移动端跨平台极客
* **画面流程**：
  1. [屏幕录制] 用 VS Code 打开 `apps/flutter/` 目录，控制台实录 `flutter pub get` 安装进度。
  2. [切真机/抓手视界] 连接一部 Android 13 以上的手机，点击运行按钮！
  3. [讲干货] 在控制台指出我们写好的 `permission_handler`，展示一旦拒绝权限就搜不到设备的恶心景象，随后现场指点迷津，给权限后瞬间爆搜满屏周围红颜色的外设备。

## 📹 [VLog 02] 微信小程序降服战：UniApp 真机神颜展示
* **主旨对象**：国内最庞大的 Vue / 微信前端开发圈
* **画面流程**：
  1. [屏幕录制] 双击打开那个绿色的魔法棒 **HBuilderX** 编辑器，把整个 `apps/uniapp` 甩进去拉开工程。
  2. [操作重灾区] 点击“真机运行到微信开发者工具”，重点给开发者指出如何看我们里面 `MAC 地址替换成 UUID` 机制的绕坑代码片段（因为这可是连百度都很难直接搜到的大坑）。
  3. [炫技] 不要多废话，在这个大工具开着的时候，随便在旁边的 Terminal 里面跑一次我们的 `python generate_assets.py`。让开发者看着小程序自动重载变装暗黑魔法主题！

## 📹 [VLog 03] Tauri vs Electron：性能与极致轻量的神仙角斗场
* **主旨对象**：React / JS Node 玩家，Rust 痴迷者，桌面生产力大王
* **画面流程**：
  1. [极限分屏] 屏幕切分！左侧打开 `apps/desktop/tauri`，右侧打开 `apps/desktop/electron`。
  2. 左下角的 CMD 开始跑 C++ Node 构建（满屏眼花缭乱警告），右下角的 CMD 开始跑 Rust 编译！
  3. 分别双击两个编译出来的 `exe / msi` 包进行展示 —— 特地指明 Rust 的包仅有十几兆，而老霸主却有一百兆的魔幻对比，展示跨平台多核底座的神仙并列结构！

## 📹 [VLog 04] 原生双煞挑战：XCode 与 Android Studio 齐开
* **主旨对象**：崇向不套壳的原生态巨佬
* **画面流程**：
  1. 暴力美学！在极其高配的 Macbook 上，左半边大发神威挂起最重的地狱堡垒 **XCode**，右半边拉扯起内存吞噬怪兽 **Android Studio**。
  2. 分别把 `apps/ios/` 与 `apps/android/` 工程吃进来。
  3. 分发给一排桌子上的真机测试群：Swift 包发往左侧的苹果全家桶集群（iPad / iPhone）；Kotlin 代码同时并发打包，直冲右侧一整套 Android 真机设备！在真实机台上实现蓝牙互殴点灯测试！高潮下播。
